The Jennifer Benchley stories

 

These are set in Cambridge, UK, were written in the late 70s and looked forward to the late 80s.

Minimal violence, no sex (but some implications) or swearing. There is reference to “real” magic e.g. Tarot readings. Accessibility issue: the first story contains some purely decorative images of the Tarot readings described in the text. The cards are laid out in tables so as to reduce the image download size; browsing in a text-only browser will be interrupted by these decorations. I'd suggest a 12 rating at worst.
